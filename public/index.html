TLD API

Usage Instructions:

To check if a TLD is valid, use:

GET /is-a-tld?tld=<tld>

Replace <tld> with the TLD you want to check.

Example: GET /is-a-tld?tld=com


To check if a domain contains a TLD, use:

GET /contains-tld?link=<domain>

Replace <domain> with the domain you want to check.

Example: GET /contains-tld?link=example.com
